# 개요 

6장을 포스팅한 뒤 갑자기 13장을 포스팅하게 되었습니다. 

앞부분에서 TDD 가 무엇인지를 파악한 뒤 필요한 부분 부터 보려고 하다 보니까 13장을 우선적으로 보게 되었습니다. 


# 레거시 코드 변경 정책 

* 새로운 코드는 테스트 주도로 개발한다
* 레거시 코드를 수정하기 전에 테스트를 추가한다 
* 레거시 코드에서 변경되는 것들은 테스트 주도로 개발한다
