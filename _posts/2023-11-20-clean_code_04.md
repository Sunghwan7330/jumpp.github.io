```
나쁜 코드에 주석을 달지 마라. 새로 짜라 
- 브라이언 W. 커니핸, P. J. 플라우거
```

## 개요

- 주석은 오래될수록 코드에서 멀어짐
    - 프로그래머들이 주석을 유지하고 보수하기는 현실적으로 어려움
- 부정확한 주석은 아예 없는 것보다 안좋음
